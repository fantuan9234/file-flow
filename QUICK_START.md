# FileFlow 快速使用指南

## 🎯 重构后的新功能

### 统一的重命名规则卡片

所有 5 种重命名规则现在整合在一个卡片中，通过下拉框切换：

1. **添加前缀** - 在文件名前添加文本
2. **添加后缀** - 在文件名后添加文本
3. **查找替换** - 查找并替换文件名中的文本
4. **插入日期** - 在文件名中插入当前日期
5. **添加序号** - 为文件添加递增序号

---

## 📖 使用步骤

### 步骤 1：选择文件夹
点击 **"选择文件夹并扫描"** 按钮，选择要重命名的文件所在文件夹。

### 步骤 2：选择规则类型
在 **"重命名规则"** 卡片中，从下拉框选择规则类型。

### 步骤 3：配置参数
根据选择的规则，在参数输入区填写相应参数。

### 步骤 4：预览效果
点击 **"预览效果"** 按钮，查看重命名结果。

### 步骤 5：执行或撤销
- **执行重命名** - 应用重命名操作
- **撤销** - 撤销上一次操作

---

## 🔧 规则详解

### 1. 添加前缀

**用途**：在文件名前面添加文本

**参数**：
- 前缀：要添加的文本

**示例**：
```
前缀：img_
原文件：photo.jpg
结果：img_photo.jpg
```

---

### 2. 添加后缀

**用途**：在文件名后面添加文本

**参数**：
- 后缀：要添加的文本

**示例**：
```
后缀：_v2
原文件：document.txt
结果：document_v2.txt
```

---

### 3. 查找替换

**用途**：查找并替换文件名中的文本

**参数**：
- 查找：要查找的文本
- 替换为：替换成的文本

**示例**：
```
查找：old
替换为：new
原文件：old_file.txt
结果：new_file.txt
```

---

### 4. 插入日期

**用途**：在文件名中插入当前日期

**参数**：
- 位置：日期在前或在后
- 格式：日期格式字符串

**常用格式**：
- `yyyyMMdd` → 20250130
- `yyyy-MM-dd` → 2025-01-30
- `yyyy 年 MM 月 dd 日` → 2025 年 01 月 30 日

**示例**：
```
位置：前面
格式：yyyyMMdd
原文件：report.docx
结果：20250130report.docx
```

---

### 5. 添加序号

**用途**：为文件添加递增序号

**参数**：
- 起始数字：序号从几开始（默认 1）
- 步长：每次增加多少（默认 1）
- 位数：序号占几位，不足补零（默认 3）
- 位置：序号在前或在后

**示例 1**：基本序号
```
起始：1, 步长：1, 位数：3, 位置：前面
原文件：file1.txt, file2.txt, file3.txt
结果：001_file1.txt, 002_file2.txt, 003_file3.txt
```

**示例 2**：自定义起始
```
起始：10, 步长：1, 位数：2, 位置：前面
原文件：doc1.pdf, doc2.pdf
结果：10_doc1.pdf, 11_doc2.pdf
```

**示例 3**：自定义步长
```
起始：0, 步长：10, 位数：3, 位置：后面
原文件：data1.xlsx, data2.xlsx
结果：data1_000.xlsx, data2_010.xlsx
```

## 💡 实用技巧

### 技巧 1：参数保留
切换规则类型时，之前填写的参数会自动保留，方便快速切换对比效果。

### 技巧 2：重置参数
点击卡片右上角的 **"重置参数"** 按钮，一键恢复所有默认值。

### 技巧 3：扩展名保护
所有规则都不会改变文件扩展名，只修改文件名主体。

### 技巧 4：序号分隔符
添加序号时，自动使用 `_`（下划线）连接序号和文件名。

---

## ⚠️ 注意事项

1. **预览优先**：执行重命名前务必先预览，确认无误后再执行
2. **撤销功能**：支持撤销上一次重命名操作，但不支持多次撤销
3. **参数验证**：查找替换必须填写查找文本，插入日期必须填写格式
4. **文件顺序**：序号按照文件列表的顺序生成

## 🚀 快速上手

### 场景 1：批量添加前缀
1. 选择文件夹
2. 规则类型：添加前缀
3. 输入前缀：`2025_`
4. 点击预览效果
5. 执行重命名

### 场景 2：照片排序
1. 选择文件夹
2. 规则类型：添加序号
3. 设置：起始=1, 步长=1, 位数=3, 位置=前面
4. 点击预览效果
5. 执行重命名

### 场景 3：版本更新
1. 选择文件夹
2. 规则类型：查找替换
3. 查找：`v1`, 替换为：`v2`
4. 点击预览效果
5. 执行重命名

### 场景 4：添加日期标记
1. 选择文件夹
2. 规则类型：插入日期
3. 位置：前面，格式：`yyyy-MM-dd`
4. 点击预览效果
5. 执行重命名
